Описание проекта

Проект был создан для соревнований ROS2 роботов, проходящих в рамках ROS Meetup '26.Подробнее с регламентом модно ознакомиться здесь.
На соревнованиях перед участниками была поставлена цель: cоздать робота, работающего на ROS2, способного классифицировать объекты на известном полигоне и перевозить их в определенную зону. Для выполнения поставленной цели можно было пользоваться несколькими из разрешенных устройств на полигоне, около или над полигоном, на роботе:
1. Робот
2. Модуль на перекладине
3. Ноутбук рядом с полем
4. Камеру на роботе
Наша команда выбрала комбинацию из робота, модуля на перекладине и ноутбука рядом с полем.

Нами был создан робот имеющий два контроллера: микрокомпьютер Raspberry Pi 4 и контроллер Arduino UNO на основе микроконтроллера ATmega328P. К контроллерам подключены: гироскоп, сервомоторы двухзвенного манипулятора, концевые выключатели, лидар. Робот имеет дифференциальный привод на основе двух коллекторных моторов с энкодерами. Raspberry Pi обменивается данными с ноутбуком вне полигона, по Wi-fi и с Arduino UNO по протоколу UART.
На ноутбуке обрабатывается изображение с камеры, висящей над полигоном. На микрокомпьютер приходят данные о том, какие объекты стоят на каких местах и и в какой точке пролигона в данный момент находится робот. Далее программа на Raspberry в зависимости от расположения объектов, просчитывает маршрут из текущего положения робота в положение близкое к самому дорогому по баллам из ближайших объектов. Программа отсылает на Arduino обновленные значения скоростей для моторов. Когда робот доезжает до заданной точки, робот начинает стыковку с объектов с помощью лидара, а далее, когда стыковка завершена, осталает на Arduino команду о захвате объекта. После этого робот отправляется на точку выгрузки объектов, после чего начинает поиск нового объекта.
