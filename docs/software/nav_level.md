# Описание алгоритмов, работающих на уровне компьютера с камерой над Charuco-доской

```
┌──────────── Изображение с камеры
│               │              │
│               ▼              ▼
│  ┌───────────────────┐ ┌───────────────────┐
│  │Определение позиции│ │Определение позиции│
│  │   charuco-доски   │ │   aruco-маркера   │
│  └────┬───────┬──────┘ └────┬──────────────┘
│       │       │             │  
│       │       ▼             ▼
│       │     ┌──────────────────┐
│       │     │Вычисление позиции│
│       │     │ маркера на доске │
│       │     └──────────────────┘
│       │               │
│       │               ▼
│       │      Отправка на робота
▼       ▼
┌───────────────────┐
│"Выпрямление" доски│
│ по известной поз. │
└───────────────────┘
         │
         ▼
┌───────────────────┐
│  Определение  16  │
│ стартовых позиций │
└───────────────────┘
         │
         ▼
┌───────────────────┐
│Определение объекта│
│    в  каждой из   │
│ стартовых позиций │
└───────────────────┘
         │
         ▼
 Отправка на робота
```
