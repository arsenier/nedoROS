# Электроника проекта

С логикой подключения электроники можно ознакомиться на схеме Э3 ниже.
Arduino UNO вместе с драйвером L298 управляют моторами. Arduino также обрабатывает показания с концевиков, MPU и энкодеров, управляет пищалкой и сервоприводами.
![alt text](image.png)

Связь Arduino и Raspberry Pi осуществляется кабелем USB-B - USB-A. Важно, что на стороне USB-A, подключаемой к Raspberry Pi, изолирован контакт 5 В. В нашем случае это сделано изолентой.

К Raspberry Pi также подключаются камера и лидар.
![alt text](image-1.png)

Питание осуществляется от 3S-аккумулятора. Для Arduino отдельный стабилизатор не используется. LM2596 используется для питания сервоприводов, а XL4015 - для Raspberry Pi. Аварийная кнопка отключает только питание Arduino и разрывает цепь, входящую в шилд.
